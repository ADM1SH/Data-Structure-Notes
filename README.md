# CCP4213 - Data Structures & Algorithms Revision Guide

This folder contains a collection of C++ files organized by topic, designed to help you revise for your final exam. The code is based on the lecture notes, lab solutions, and the final exam papers from February and July 2025.

Each `.cpp` file includes:
1.  **Core Logic**: General implementations of data structures (like a Stack class or a Linked List).
2.  **Exam Snippets**: Specific, isolated functions that solve questions directly from your past exam papers.

---

## Folder Structure & Exam Snippets Coverage

### 1. `Topic02_Pointers/`
- **`pointers_revision.cpp`**:
  - **Exam Snippet (Feb 2025)**: Full pointer tracing logic.
  - **Exam Snippet (July 2025)**: Full pointer tracing logic with reference variables.
  - Covers pointer arithmetic, dereferencing, and dynamic memory (`new`/`delete`).

### 2. `Topic03_05_Stacks_Queues/`
- **`stacks_queues_revision.cpp`**:
  - **Exam Snippet (Feb 2025)**: Completes the array-based `pop()` function definition.
  - **Exam Snippet (Feb 2025)**: Provides a commented trace of the complex Stack/Queue operations involving variables `S`, `Q`, `X`, and `Y`.
  - Includes both array-based and linked-list-based implementations.

### 3. `Topic04_LinkedLists/`
- **`linked_lists_revision.cpp`**:
  - **Exam Snippet (Feb 2025)**: Code segment to `total up all even numbers` in a list.
  - **Exam Snippet (Feb 2025)**: Code segment to `swap the data value` of the first and last nodes.
  - **Exam Snippet (July 2025)**: Code segment to `add a new node` between two existing nodes.
  - **Exam Snippet (July 2025)**: Code segment to `delete the node` with specific data.

### 4. `Topic06_07_Searching_Hashing/`
- **`searching_hashing_revision.cpp`**:
  - **Exam Snippet (Feb 2025)**: Step-by-step logic for **Binary Search**.
  - **Exam Snippet (Feb 2025)**: Hashing by **truncation** function.
  - **Exam Snippet (Feb 2025)**: Commented trace of **Linear Probing** collision resolution.

### 5. `Topic08_Sorting/`
- **`sorting_revision.cpp`**:
  - **Exam Snippet (Feb 2025)**: **Selection Sort** implementation (descending) with pass-by-pass tracing.
  - Covers the "Divide and Conquer" concept of **Merge Sort**.

### 6. `Topic09_Trees/`
- **`trees_revision.cpp`**:
  - **Exam Snippet (Feb 2025)**: Explanation and structural breakdown for drawing an **Algebraic Expression Tree** (`J / (M + N) - K * L`).
  - Includes PreOrder, InOrder, and PostOrder traversal functions.

### 7. `Topic10_Graphs/`
- **`graphs_revision.cpp`**:
  - **Exam Snippet (Feb 2025)**: Explanation of how to represent a graph using an **Adjacency Matrix**.
  - **Exam Snippet (Feb 2025)**: Logic for identifying cycles.
  - Covers high-level concepts for **BFS** and **DFS** traversals.
